# NEXUS AI Manager

## Setup (100% Free Stack)
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
Open: http://localhost:8000

## Login Credentials
- **Admin:** admin@nexus.ai / admin123
- **Demo User:** demo@nexus.ai / demo123

## Features
- Customer dashboard with AI briefing
- AI Chat (Gemini 1.5 Flash)
- Calendar, Tasks (Kanban), Habits, Goals, Notes, Budget
- Analytics with charts
- Billing & Plan management
- Admin panel (user management, billing overview)

## Tech Stack
- FastAPI (Python) + SQLite
- Jinja2 templates + Chart.js
- Google Gemini AI (free)
- JWT authentication
